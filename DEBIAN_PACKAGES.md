# Debian Packages

This repository contains the source code for LTFS. For Debian packages, please see:

https://github.com/Mikec78660/ltfs-debian-packages

## Available Packages

- `ltfs_2.5.0.0-1_amd64.deb` - Main LTFS package
- `ltfs-dev_2.5.0.0-1_amd64.deb` - Development package

## Installation

```bash
# Download from the Debian packages repository
wget https://github.com/Mikec78660/ltfs-debian-packages/releases/download/v2.5.0.0-1/ltfs_2.5.0.0-1_amd64.deb
sudo dpkg -i ltfs_2.5.0.0-1_amd64.deb
```
